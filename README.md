# shooter-simulator-game
###The game doesn't have any place to show you the controls in game.
###Controls
###--------
###'o' aim down the sights
###'p' fire weapon
###'r' reload weapon
###'g' grenade
###'w' forward movement
###'a' left movement
###'s' back movement
###'d' right movement
###'1' first weapon
###'shift' sprint
###'c' crouch
